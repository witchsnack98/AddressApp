# Address Management — Ubiquitous Language

## Core Terms

| Term | Definition |
|------|------------|
| **Address** | A postal location record belonging to a person, identified by a UUID. |
| **firstName** | Given name of the person associated with the address. |
| **lastName** | Family name of the person associated with the address. |
| **houseNumber** | บ้านเลขที่/ซอย — building number and/or soi (alley). |
| **street** | ถนน — road or street name. |
| **subDistrict** | ตำบล/แขวง — sub-district or khwaeng. |
| **district** | อำเภอ/เขต — district or khet. |
| **province** | จังหวัด — province. |
| **postalCode** | รหัสไปรษณีย์ — Thai postal code (5 digits). |
